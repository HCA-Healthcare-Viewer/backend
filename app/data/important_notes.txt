Important Segments:

	MSH (Message Headers)

		MSH-7 	: Date/Time of Message (used for sorting messages)
		MSH-9	: Message Type (used for summary/preview)
		MSH-10 	: Message Control ID (used for summary/preview and filtering)


	PID (Patient ID Info.)


		PID-3	: Patient ID List, MRN (used for summary/preview and filtering)
		PID-5	: Patient Name

			PID-5.1 	: Last Name
			PID-5.2		: Given Name
			PID-5.7 	: Middle Initial
